# Heart Disease Prediction using Machine Learning

## 📌 Overview
This project predicts heart disease using ML models.

## 📊 Dataset
- UCI Heart Disease Dataset
- 303 patient records
- 14 clinical features

## ⚙️ Models Used
- Decision Tree
- Random Forest
- KNN (Best: 90% accuracy)

## 📈 Results
- Accuracy: 90%
- Precision: 0.95
- Recall: 0.79
